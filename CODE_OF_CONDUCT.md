# Code of conduct

## The short version

Be kind. Be respectful. Assume good intent. This project is a place to build and learn, not to tear people down.

## Our standards

Things that help everyone:

- Use welcoming and inclusive language.
- Respect different viewpoints and experiences.
- Accept constructive feedback with grace.
- Focus on what is best for the project and the people in it.

Things that are not welcome:

- Harassment, insults, or personal attacks.
- Discriminatory jokes or language.
- Publishing someone's private information without their consent.
- Any conduct that would be inappropriate in a professional setting.

## Scope

This applies in all project spaces: issues, pull requests, discussions, and any public space where you
represent the project.

## Reporting

If you see or experience unacceptable behavior, report it to Sabahudin Murtic on LinkedIn:
https://www.linkedin.com/in/sabahudin-murtic/ . Reports are handled privately and taken seriously.
